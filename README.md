# 🛒 Raven POS — Supermarket Point of Sale (Static Demo)

A fully functional **frontend-only** Point of Sale (POS) interface designed for supermarkets and retail stores.  
This is a demonstration project showcasing a clean UI, real‑time inventory management, sales simulation, and low‑stock alerts – all powered by **localStorage** (no backend, no database).

---

## 🚀 Features

- **Product Management** – Add, edit, delete products with name, price, stock, and category.
- **Inventory Tracking** – Stock updates automatically on each sale (saved in localStorage).
- **Low Stock Alerts** – Visual warning when stock falls below a threshold.
- **Sales Dashboard** – Daily, weekly, and monthly sales summaries (simulated).
- **Customer Management** – Record customer names and purchase history (localStorage).
- **Order Processing** – Complete checkout flow with receipt preview.
- **Admin Dashboard** – Full control panel for store owners.
- **Responsive Design** – Works on desktop, tablet, and mobile.

---

## 🛠️ Tech Stack

| Layer        | Technology                         |
|--------------|------------------------------------|
| **Frontend** | HTML5, CSS3, Vanilla JavaScript    |
| **Data**     | localStorage (browser‑side storage)|
| **Deployment**| GitHub Pages (static hosting)     |
| **Icons**    | Font Awesome (optional)            |

> ⚠️ **This is a demo project** – no backend, no authentication, no real payment processing. All data resets when browser cache is cleared.
